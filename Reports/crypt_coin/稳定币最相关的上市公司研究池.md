# 最相关的上市公司研究池（稳定币价值链）

> 目标：用于买方“公司调研”初筛。
>
> 排序口径（可复核、便于讨论）：**利润池大小 × 控制力**。
> - **控制力** = 网络效应（多边网络/默认通道） + 企业工作流嵌入程度（替代成本/切换摩擦）。
> - **不把牌照当作“稀缺护城河”**：但在现实中，合规落地仍需要满足监管要求（许可、储备、披露、AML/KYC、托管等）。
>
> 使用提示：本清单仅列“已上市公司”，且以“公开披露/公告/财报/监管材料”为依据；后续调研请以最新披露为准并补齐定量数据（流通量、交易量、收入结构、费率、渗透率等）。

## 排序列表（由高到低）

### 1) Circle Internet Financial（NYSE: CRCL）
- **环节**：稳定币发行 / 申赎通道 / 网络运营（USDC）
- **利润模式**：储备收益（利率相关）+ 机构申赎/企业结算相关收入 + 生态合作分成（视协议）
- **控制力来源**：
  - 网络效应：USDC 作为主流美元稳定币之一，跨交易所/链/机构合作广泛
  - 嵌入程度：向机构申赎、支付网络、金融基础设施渗透（更贴近 To B）
- **当前进展（需核对最新）**：USDC 多链发行、与支付网络/金融科技/银行体系合作扩展
- **跟踪指标**：USDC 流通量与份额、储备结构与披露频率、储备收益与分成结构、To B 结算/出金合作落地
- **信息来源类型**：公司公告/招股书与财报、主要合作方公告、监管与审计/鉴证披露

### 2) Coinbase（NASDAQ: COIN）
- **环节**：交易所与分发入口 / 流动性与做市生态 / USDC 重要合作方
- **利润模式**：交易与托管相关收入 + 与稳定币相关的分成/收益（取决于协议与平台持有结构）
- **控制力来源**：
  - 网络效应：面向零售与机构的主流分发入口之一，稳定币是其交易与资金管理“底座资产”
  - 嵌入程度：逐步向商户收款/支付产品、链上结算入口延伸（但企业工作流嵌入仍需验证）
- **当前进展（需核对最新）**：围绕 USDC 的支付/商户与链上结算产品推进；与发行方关系结构可能随时间调整
- **跟踪指标**：稳定币相关收入拆分、平台内稳定币余额、商户/支付产品采用、机构资金流入与留存
- **信息来源类型**：财报与电话会、公司公告、合作伙伴公告

### 3) Visa（NYSE: V）
- **环节**：支付网络 / 清算与结算编排（将稳定币用于结算轨道）
- **利润模式**：网络服务费（结算与交易量相关）+ 向银行/收单输出产品能力的服务费（视产品）
- **控制力来源**：
  - 网络效应：全球多边网络（发卡行—收单—商户—持卡人）天然“默认通道”属性
  - 嵌入程度：深度嵌入银行与商户清结算工作流，替代成本高
- **当前进展（需核对最新）**：稳定币结算试点、与收单/银行合作扩展；银行链上资产/代币平台相关布局
- **跟踪指标**：稳定币结算合作覆盖（银行/收单）、可量化的结算量披露、产品化进度与商业条款
- **信息来源类型**：公司公告/投资者材料、合作伙伴公告

### 4) Mastercard（NYSE: MA）
- **环节**：支付网络 / 合规与链上风控能力（地址筛查等）/ 多资产结算探索
- **利润模式**：网络费 + 合规风控/数据与服务费 + 多资产网络产品化收入（若落地）
- **控制力来源**：
  - 网络效应：与 Visa 类似的全球多边网络地位
  - 嵌入程度：深度嵌入发卡/收单/商户体系；并尝试把链上合规能力产品化
- **当前进展（需核对最新）**：稳定币相关合作、合规风控能力对接、企业/机构支付试点
- **跟踪指标**：稳定币结算覆盖与合作伙伴、合规产品收入与渗透、企业端用例落地
- **信息来源类型**：公司公告、合作伙伴公告

### 5) PayPal（NASDAQ: PYPL）
- **环节**：支付与钱包分发入口 / 自有稳定币（PYUSD）品牌与生态
- **利润模式**：支付与资金管理收入 + 稳定币相关储备收益/服务费（视结构）
- **控制力来源**：
  - 网络效应：庞大的线上支付与钱包网络
  - 嵌入程度：对个人与商户支付工作流嵌入强；To B 渗透取决于其商户与Braintree生态推进
- **当前进展（需核对最新）**：PYUSD 在 PayPal/Venmo 体系内的使用拓展；与支付处理/银行基础设施合作
- **跟踪指标**：PYUSD 流通量与活跃度、商户/收单支持、合规与赎回通道、与核心支付业务协同
- **信息来源类型**：公司公告、发行/托管方披露、财报

### 6) Fiserv（NASDAQ: FI）
- **环节**：支付处理与核心银行系统（企业/银行工作流）/ 自有稳定币或稳定币能力集成（如 FIUSD）
- **利润模式**：软件/处理费 + 可能的稳定币相关服务费与生态分成（视产品）
- **控制力来源**：
  - 网络效应：连接大量银行与商户的支付处理网络
  - 嵌入程度：深度嵌入银行核心系统与支付后台，替代成本高
- **当前进展（需核对最新）**：推出稳定币相关计划并与行业伙伴合作；将能力嵌入自身平台
- **跟踪指标**：银行客户开通数量、稳定币处理量、产品商业条款与毛利结构
- **信息来源类型**：公司公告/投资者材料、合作伙伴公告

### 7) FIS / Worldpay（NYSE: FIS）
- **环节**：收单与商户结算 / 银行资金流与支付中台（稳定币结算与出金选项）
- **利润模式**：收单与结算费 + 向银行输出稳定币能力的中台服务费
- **控制力来源**：
  - 网络效应：商户收单网络与银行客户网络
  - 嵌入程度：商户对账结算与银行资金流系统嵌入深
- **当前进展（需核对最新）**：与稳定币发行方/支付网络合作探索稳定币结算；对银行输出相关能力
- **跟踪指标**：开通稳定币结算商户数量、结算量、对银行客户的覆盖与产品化程度
- **信息来源类型**：公司公告、合作伙伴公告

### 8) JPMorgan Chase（NYSE: JPM）
- **环节**：企业资金调拨与跨境结算（存款代币/链上支付网络）
- **利润模式**：企业银行与支付服务收入 + 提升资金效率带来的客户留存与增量交易
- **控制力来源**：
  - 网络效应：强大的企业银行客户基础与同业网络
  - 嵌入程度：深度嵌入企业财资工作流（资金池、跨境调拨、结算）
- **当前进展（需核对最新）**：存款代币/链上结算平台持续扩展用例（回购、抵押品等）
- **跟踪指标**：链上结算量、参与企业与同业数量、用例扩展与商业条款
- **信息来源类型**：公司公告/投资者材料、财报

### 9) BNY Mellon（NYSE: BK）
- **环节**：储备托管与资产服务（稳定币储备托管、机构托管）
- **利润模式**：托管费/服务费 + 机构客户资金沉淀带来的综合收益
- **控制力来源**：
  - 网络效应：托管银行的“信任网络”与机构连接
  - 嵌入程度：深度嵌入机构资产托管、合规与报告工作流
- **当前进展（需核对最新）**：参与稳定币储备托管与数字资产业务布局
- **跟踪指标**：托管规模、相关客户数量、数字资产托管收入贡献
- **信息来源类型**：公司公告、合作伙伴披露

### 10) BlackRock（NYSE: BLK）
- **环节**：稳定币储备资产管理（短债/货币基金等）
- **利润模式**：AUM 管理费（费率低但规模大）+ 相关产品规模扩张
- **控制力来源**：
  - 网络效应：资产管理行业龙头与机构分发网络
  - 嵌入程度：机构资金管理与配置工作流（合规偏好强）
- **当前进展（需核对最新）**：与头部稳定币生态在储备资产管理层面存在合作/产品连接
- **跟踪指标**：相关基金AUM、费率与收入贡献、与稳定币生态的合作范围扩展
- **信息来源类型**：基金披露/公司公告、合作伙伴披露

### 11) SAP（NYSE: SAP / XETRA: SAP）
- **环节**：企业ERP/财资工作流集成（稳定币支付与对账）
- **利润模式**：企业软件订阅与模块/实施服务费（“稳定币能力”成为新卖点）
- **控制力来源**：
  - 网络效应：ERP 生态与大型企业客户群
  - 嵌入程度：极强（财务/付款/对账在ERP里），替代成本高
- **当前进展（需核对最新）**：数字货币/链上支付相关模块与试点
- **跟踪指标**：试点客户转化、稳定币支付笔数/金额（若披露）、模块收入贡献
- **信息来源类型**：产品文档/公司公告、客户案例

### 12) Société Générale（EPA: GLE）
- **环节**：银行系稳定币发行（机构端）/ 交易与结算用例
- **利润模式**：机构发行/兑换与结算服务费 + 与机构客户的综合金融收入
- **控制力来源**：
  - 网络效应：银行客户网络与交易对手网络
  - 嵌入程度：对机构结算与资金调拨工作流具备嵌入潜力（需用数据验证）
- **当前进展（需核对最新）**：通过子公司/平台推进欧元与美元稳定币/代币化工具，用于机构结算
- **跟踪指标**：流通量、参与机构数量、结算/抵押品用例落地
- **信息来源类型**：公司公告、监管披露、合作伙伴公告

### 13) Deutsche Börse（XETRA: DB1）
- **环节**：市场基础设施 / 交易后服务（抵押品、清算、结算）
- **利润模式**：清算结算服务费 + 新的链上抵押品/结算产品收入
- **控制力来源**：
  - 网络效应：交易所与清算基础设施的强网络效应
  - 嵌入程度：机构交易后工作流嵌入极强，替代成本高
- **当前进展（需核对最新）**：探索/试点使用（合规）稳定币作为抵押品与结算资产
- **跟踪指标**：试点产品落地、参与成员数量、链上结算占比
- **信息来源类型**：公司公告、行业/合作伙伴公告

### 14) Western Union（NYSE: WU）
- **环节**：跨境汇款网络 / 现金出入金（off-ramp）
- **利润模式**：汇款费率与外汇点差 + 若引入稳定币可改善成本结构并增强网络粘性
- **控制力来源**：
  - 网络效应：线下代理与跨境支付网络
  - 嵌入程度：对“现金—数字”最后一公里具备渠道优势
- **当前进展（需核对最新）**：披露过稳定币/数字资产网络相关计划（以公告为准）
- **跟踪指标**：数字资产网络覆盖、稳定币相关产品上线与规模、成本率变化
- **信息来源类型**：公司公告、合作伙伴公告

### 15) Robinhood（NASDAQ: HOOD）
- **环节**：零售经纪与钱包入口（稳定币交易/持有）
- **利润模式**：交易相关收入 + 资金/利息相关收入（视产品结构）
- **控制力来源**：
  - 网络效应：零售用户入口
  - 嵌入程度：对个人资金管理有一定嵌入，但企业工作流嵌入较弱
- **当前进展（需核对最新）**：支持稳定币交易/钱包等能力
- **跟踪指标**：稳定币交易量、用户稳定币余额、相关收入拆分
- **信息来源类型**：财报与产品公告

### 16) Standard Chartered（LSE: STAN）
- **环节**：跨境银行网络 / 多银行结算平台参与（代币化存款/链上结算方向）
- **利润模式**：跨境支付与交易银行业务增量 + 平台参与带来的潜在服务收入
- **控制力来源**：
  - 网络效应：跨境网络与同业合作平台
  - 嵌入程度：嵌入企业跨境支付与资金管理工作流（需数据验证）
- **当前进展（需核对最新）**：参与多银行链上结算相关平台/试点
- **跟踪指标**：平台规模、参与机构数量、处理交易量、商业化条款
- **信息来源类型**：公司公告、平台与合作伙伴披露

## 备注：关于“牌照是否稀缺”
- **结论（用于研究框架）**：牌照本身不应当被简单视为护城河；但**可落地的合规能力**（治理、储备管理、披露、风控、银行关系、审计、对接企业系统）在现实中往往体现为“时间成本+系统成本”，并且在大规模 To B 采用阶段会显著影响速度与客户信任。
- **调研建议**：对每家公司单独核查：适用监管框架、是否具备机构主申赎/托管/对账能力、以及被监管与客户要求倒逼的合规成本。
